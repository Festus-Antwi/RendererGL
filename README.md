# RendererGL

RendererGL is a basic 3D engine written in C++ and OpenGL

The file

```
main.cpp
```

is a template using ImGui and RendererGL

# Dependencies
```
GLFW
GLEW (included)
GLM (included)
IMGUI (included)
ASSIMP
```
