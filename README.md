# RendererGL

RendererGL is a basic 3D renderer in Modern OpenGL and C++

```
main.cpp
```

is a template using ImGui and RendererGL

# Dependencies
```
GLFW
GLEW (included)
GLM (included)
IMGUI (included)
ASSIMP
```
